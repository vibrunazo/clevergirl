How does it work? The game generate bot names automatically from a list found in the game APK. You can see the entire list from the original file here. Bot names are always composed from a combination of a random adjective + a random name from that list, there is never an exception to this rule.

This app simply looks up if the player name that you typed matches that pattern. The autocomplete from the input will suggest names from that pre-defined list.
