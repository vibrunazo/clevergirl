Welcome

To Clever Girl

Clever Girl is a fan site for the game Jurassic World: Alive, by Ludia. Development for Clever Girl started June 28 2018, and the first public launch was on July 10, 2018.

You can use the Battle Simulator to simulate fights between every Dinosaur in the game. The simulator should be fully functional. The Simulator let's you fight battles against yourself or against bots. You can also define how luck will play out in the simulation. So you can use it to test exactly what you need to help you decide what dinosaurs to level up and use on your team in the actual game. If you find any problems please let me know.

Clever Girl has updated stats for all dinosaurs in the game on the stats page. Here you can also check out if a player you just battled against was a Bot or Not.

Clever Girl was built using Angular and Angular Material. It is built and maintained by one single developer. You can reach me on the contact page. The website was initially developed as a fun educational project to learn Angular, but it has since evolved to a little more.

This website is just a simple Static web Page, it has only the front end with no back end code or a database. It will not directly collect any sort of personal identifying information about its users, and has no way of storing those. But it does use Google Analytics to understand how the website is used. And it may display ads from Google AdSense that can use information from the user behaviour elsewhere. You can read more about this on the privacy policy.

The name "Clever Girl" is a reference to a famous quote in the original Jurassic Park movie. Watch the scene on Youtube.
